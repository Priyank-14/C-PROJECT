# 🧮 Matrix Calculator in C

A menu-driven CLI application that performs fundamental matrix operations using core C programming concepts.

---

## 📌 Overview

This project implements a **Matrix Calculator** in C that performs various operations on matrices using a structured, menu-driven interface.

It demonstrates the application of **arrays, functions, loops, and conditional logic** in building a practical console-based program.

---

## 🎯 Objective

* Apply core C programming concepts in a real program
* Work with **2D arrays and matrix operations**
* Understand modular programming using functions
* Strengthen logical thinking and problem-solving

---

## ✨ Features

The program supports:

1. Matrix Addition
2. Matrix Subtraction
3. Matrix Multiplication
4. Transpose of a Matrix
5. Scalar Multiplication
6. Trace of a Matrix
7. Check Symmetric Matrix
8. Check Identity Matrix
9. Determinant (2×2 and 3×3)
10. Exit

---

## 🛠️ Tech Stack

* Language: C
* Concepts: Arrays, Functions, Loops, Conditionals

---

## ▶️ How to Run

```bash
gcc Project.c -o matrix
./matrix
```
